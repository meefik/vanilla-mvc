## libux

A minimalistic UX framework for libraries in pure JavaScript without dependencies.

### How to run

Run the sample app [to-do list](http://todomvc.com/) and the documentation:

- run `npm install`
- run `npm start`
- open [http://localhost:8080](http://localhost:8080/)

### Run lint

```
npm run lint
```

### Build to production

```
npm run build
```
